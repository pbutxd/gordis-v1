# 🤖 Gordis V-1: Tu Asistente de Cálculos Interactivo

¡Bienvenido a Gordis V-1! Soy un asistente de consola diseñado para ayudarte con una variedad de cálculos matemáticos básicos, funciones avanzadas y problemas de física/ingeniería. Mi objetivo es hacer que los cálculos sean más accesibles y, ¡ahora un poco más conversacionales!

## ✨ Características

* **Operaciones Matemáticas Básicas:** Suma, Resta, Multiplicación, División, Potencia, Raíz Cuadrada, Módulo.
* **Funciones Matemáticas Avanzadas:** Seno, Coseno, Tangente (en grados), Logaritmo Natural, Logaritmo Base 10, Factorial.
* **Cálculos de Física/Ingeniería:** Ley de Ohm, Presión, Velocidad/Distancia/Tiempo, Fuerza, Energía (E=mc²), Conversión de Velocidad (m/s <-> km/h).
* **Interfaz de Usuario Amigable:** Menús claros y uso de colores para una mejor experiencia.
* **Modo Conversacional Básico:** ¡Puedes intentar escribir lo que quieres hacer! Por ejemplo, "quiero sumar", "seno de 45", "ley de ohm". Si no entiendo, te guiaré al menú.

## 🚀 Cómo Empezar

Para ejecutar Gordis V-1 en tu máquina, sigue estos pasos:

1.  **Clona el repositorio** (o descarga los archivos directamente):
    ```bash
    git clone [https://github.com/tu-usuario/nombre-del-repositorio.git](https://github.com/tu-usuario/nombre-del-repositorio.git)
    cd nombre-del-repositorio
    ```
    (Reemplaza `tu-usuario` y `nombre-del-repositorio` con los tuyos).

2.  **Crea un entorno virtual (recomendado):**
    ```bash
    python -m venv venv
    # En Windows:
    venv\Scripts\activate
    # En macOS/Linux:
    source venv/bin/activate
    ```

3.  **Instala las dependencias:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Ejecuta Gordis V-1:**
    ```bash
    python gordis_v1_avanzado.py
    ```

## 📚 Requerimientos

Gordis V-1 requiere Python 3.x. Las dependencias adicionales se listan en `requirements.txt`.
